== Description ==

The MyAnime widget leverages the engine of the MyAnime.me social
fansub monitoring site to display the latest fansubbed anime episodes
on your blog in a highly configurable fashion.

If you have a MyAnime user account, you can define the series that you
wish to follow there, and have only episodes from them displayed in
your MyAnime widget.

The widget is still in a somewhat beta state. If you have any trouble
installing or using it, do not hesitate to contact us at
support@myanime.me.


== Installation ==

1. Upload myanime.php and myanime_import_admin.php to the
/wp-content/plugins/myanime directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select location for the plugin under Appearance > Widgets
4. Customize plugin look through Settings > MyAnime
5. (Optional) Get a MyAnime.me user account and specify it in settings
to get a customized feed
